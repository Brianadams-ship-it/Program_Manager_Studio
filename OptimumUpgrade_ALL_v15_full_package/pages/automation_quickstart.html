Optimum Upgrade – Automation Quick Start Guide

Home Products Automation Hub Sample Project

This guide explains how to enable automation inside your Excel-based PM/SE toolkits, add the Control Panel, and run one-click verification workflows.

1. Requirements

  • Windows with Microsoft Excel installed
  • Macro-enabled workbook (.xlsm)
  • Excel Trust Center → Check “Trust access to the VBA project object model”

2. Importing the Automation Module

Use the included PowerShell script:

cd C:\OptimumUpgrade_Site\automation
.\import_vba.ps1

The script will:

  • Convert workbooks to .xlsm if needed
  • Inject the OptimumUpgrade_Automation.bas module
  • Save and close the workbook automatically

3. Automation Macros

  • GenerateTestCasesFromRequirements – Auto-builds Test Cases from the Requirements sheet.
  • BuildVerificationSummary – Creates or refreshes a VerificationSummary sheet.
  • ExportVerificationSummaryCSV – Exports the VerificationSummary as CSV.
  • EnsureProjectInfoSheet – Creates the ProjectInfo sheet with standard fields.
  • UpdateSheetHeadersFromProjectInfo – Updates sheet headers with project metadata.

4. Recommended Workflow

  1. Enter or import Requirements.
  2. Run Generate Test Cases.
  3. Run Build Verification Summary.
  4. Export Verification Summary (optional) using CSV.
  5. Use ProjectInfo sheet to set Program Name, Acronym, Customer, and Contract.

5. Control Panel (Optional)

You may copy the ControlPanel sheet from Automation_ControlPanel_Template.xlsx into any automation-enabled workbook and add buttons mapped to the macros.
