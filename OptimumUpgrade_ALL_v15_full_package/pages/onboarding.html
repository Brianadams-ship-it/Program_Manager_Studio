Your First 60 Minutes with Optimum Upgrade

Follow these steps to get value from the suite quickly.

1. Unzip & Explore the Folder Layout

  • Extract the ZIP into a folder such as C:\OptimumUpgrade\.
  • Note key folders: pages, automation, developer_sdk, templates, sample_project.
  • Open Product_Catalog_OptimumUpgrade.pdf to see an overview of everything included.

2. Open the Built-In Docs Site

  • In your browser open pages/index.html.
  • From there you can navigate to Products, Automation Hub, and Sample Project.

3. Run Your First Automation Flow

  • Review the Automation Quick Start PDF for Excel macro setup.
  • Use automation/import_vba.ps1 to inject the automation module into templates/Automation_Base_Toolkit.xlsx.
  • Open the resulting .xlsm file in Excel and run:
    • EnsureProjectInfoSheet
    • GenerateTestCasesFromRequirements
    • BuildVerificationSummary

4. Explore the FalconEye Sample Project

  • Open the workbooks in sample_project to see a fictional micro-UAV program.
  • Use it as a reference for how Requirements, Test Cases and Verification work together.

5. Tailor Templates for Your Program

  • Copy the starter toolkits and doc templates from templates into a new project folder.
  • Customize content and apply the formatting from Design_Spec_Office_Formatting.md.
