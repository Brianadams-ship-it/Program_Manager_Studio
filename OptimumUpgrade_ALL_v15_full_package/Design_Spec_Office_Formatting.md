# Optimum Upgrade – Office Template Formatting Specification

This specification defines how to format Word and Excel documents so they present a consistent,
professional look aligned with the Optimum Upgrade brand.

## 1. Fonts

- Primary font: **Segoe UI** (preferred) or **Arial** (fallback)
- Heading font: **Segoe UI Semibold**
- Code and technical snippets: **Consolas**

## 2. Color Palette

- Primary blue: **#0057A0** (titles, header bars, main accents)
- Accent blue: **#0093D5** (links, secondary accents)
- Text gray: **#1C1C1C** (body text)
- Soft blue: **#F4F9FF** (background for callouts and tables)
- Light gray: **#E0E0E0** (borders and subtle dividers)

## 3. Word Document Standards

### 3.1 Page Layout

- Page size: Letter (8.5 x 11 in)
- Margins: 1 inch all around
- Body text: 10–11 pt, line spacing 1.15–1.2

### 3.2 Header and Footer

- Header:
  - Left: document title (e.g., "Project Management Plan")
  - Right: project name and version (e.g., "FalconEye v1.0")
- Footer:
  - Left: company or team name
  - Right: page number (e.g., "Page 1 of 20")

### 3.3 Heading Styles

- Heading 1:
  - 16–18 pt, bold, primary blue (#0057A0)
- Heading 2:
  - 14 pt, bold, primary blue
- Heading 3:
  - 12 pt, bold, dark gray (#1C1C1C)

Use Word styles rather than manual formatting so changes can be applied consistently.

### 3.4 Tables

- Header row:
  - Fill: primary blue (#0057A0)
  - Font: white, bold
- Body rows:
  - Alternating fill: white / soft blue (#F4F9FF)
- Borders:
  - Thin gray lines (#E0E0E0)
- Alignment:
  - Text: left-aligned
  - Numbers: right-aligned when appropriate

## 4. Excel Workbook Standards

### 4.1 Top Information Banner

Reserve the top two rows for project metadata:

- Row 1: Program name and acronym (merged across key columns)
  - Background: primary blue, white bold text
- Row 2: Customer, contract number, program ID, and revision
  - Background: soft blue, dark gray text

### 4.2 Column Headers

- Background: primary blue
- Font: white, bold
- Row height: 24–28
- Freeze panes: keep the header row visible when scrolling

### 4.3 Data Rows

- Alternating row colors: white / soft blue (#F4F9FF)
- Borders: thin gray (#E0E0E0)
- Use consistent formats:
  - Dates formatted as YYYY-MM-DD or a consistent local format
  - Currency with appropriate symbol and two decimal places

### 4.4 ControlPanel Sheet

- Title: "Automation Control Panel" in primary blue
- Layout:
  - Left side: brief description of each macro
  - Right side: Excel buttons assigned to each macro
- Use callout cells (soft blue background) to provide short instructions.

## 5. Applying the Spec

1. Create a Word or Excel template with these styles applied.
2. Use that template as the basis for new documents and workbooks.
3. For existing documents, apply styles gradually starting with headings and tables.
