# Optimum Upgrade – Training Session Agenda

## Session Title
Getting Started with the Optimum Upgrade Automation Suite

## Duration
2 hours (live session or recorded webinar)

## Audience
- Project Managers
- Systems Engineers
- Test Engineers
- Technical Leads
- SBIR / R&D team members

## Learning Objectives

By the end of this session, participants will be able to:
- Describe the core components of the Optimum Upgrade suite
- Navigate the folder structure and built-in documentation
- Run the basic automation flow from Requirements → TestCases → VerificationSummary
- Understand how to use the FalconEye sample project as a reference
- Perform a short hands-on exercise using the training workbook

## Agenda

1. Welcome and Context (10 min)
   - Why Optimum Upgrade exists
   - Typical pain points it addresses

2. Tour of the Package (15 min)
   - Folder structure and key artifacts
   - HTML docs site and PDFs

3. Automation Concepts (15 min)
   - Requirements, TestCases, and VerificationSummary
   - ProjectInfo and header propagation

4. Live Automation Demo (40 min)
   - Injecting the automation module (if needed)
   - Creating ProjectInfo
   - Entering example requirements
   - Generating test cases
   - Building the verification summary
   - Exporting CSV (optional)

5. FalconEye Sample Project Walkthrough (20 min)
   - Overview of the fictional micro-UAV program
   - How artifacts map to real projects

6. Hands-On Lab (20 min)
   - Participants use the Training Exercise workbook
   - Add requirements and run automation

7. Q&A and Next Steps (10 min)
   - How to try Optimum Upgrade on a pilot project
   - How to extend or customize the templates
